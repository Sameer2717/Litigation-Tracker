Export your data to Google Sheets

Click the buttons below to download each CSV file. Then import them into Google Sheets. Your Looker Studio dashboard will be live and auto-updating from that point on.

3-step setup instructions
1
Create your Google Sheet
Go to sheets.google.com → Blank spreadsheet. Create 5 tabs and name them: MATTERS, HEARINGS, INVOICES, DOCUMENTS, NOTICES. Import each downloaded CSV into its tab (File → Import → Upload).
2
Connect Looker Studio
Go to lookerstudio.google.com → Create → Report → Add data → Google Sheets → select your file. Add charts as shown in the setup guide. Share the report link with your team.
3
Use this tool for daily entry
Add all data here (matters, hearings, invoices). When ready to sync, download the CSVs again and paste the new rows into your Google Sheet. Looker Studio updates automatically.
